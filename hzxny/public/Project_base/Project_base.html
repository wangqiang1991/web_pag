    农业基地

    循环农业园区模式借鉴生态工业园区的经验，将种植业、养殖业、农产品加工业、生物质能源业等纳入整个循环农业产业体系中，实现园区内农业的不同产业相互依存，减少农业废弃物的产生量、降低处理的成本费用、节约能源和资源，实现农业经济发展和农村生态环境保护的良性循环。

    循环科技农业

    农业实现资源节约、环境友好、可持续发展。主要有五中模式：一是循环农业园区模式。借鉴生态工业园区的经验，将种植业、养殖业、农产品加工业、生物质能源业等纳入整个循环农业体系中，实现园区内农业的不同产业相互依存，减少农业废弃物的产生量、减低处理的成本费用、节约能源和资源，实现农业经济发展和农村生态环境保护的良性循环。

    休闲农业

    农业与“三产”的有机融合。利用农业和农村资源，为居民提供观光、休闲、度假、娱乐、健身等多项服务。

    会展农业

    以会议、展览、展销、节庆、农事等活动形式为载体，通过农业会展活动，实现了产业的有机融合，拉长农业产业链，提升农业附加值，达到“举办一个展会、拉动一个产业、富裕一方农民“的目的。
